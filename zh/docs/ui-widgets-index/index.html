核心 Widget 目录

MPFlutter 允许开发者复用大部分 Flutter Widget，以下是 MPFlutter 的核心可用 Widget，以及不可用的 Widget。

所有 Material 和 Cupertino 均不可用。

Accessibility#

所有 Accessibility 组件均不可用

  • ExcludeSemantics
  • MergeSemantics
  • Semantics

Animation and Motion#

可用#

  • AnimatedAlign
  • AnimatedBuilder
  • AnimatedContainer
  • AnimatedCrossFade
  • AnimatedOpacity
  • AnimatedPositioned
  • AnimatedSize
  • AnimatedWidget
  • DecoratedBoxTransition
  • FadeTransition
  • PositionedTransition
  • RotationTransition
  • ScaleTransition
  • SizeTransition
  • SlideTransition

不可用#

  • Hero
  • AnimatedModalBarrier
  • AnimatedListState
  • AnimatedDefaultTextStyle

Assets, images, and icon#

可用#

  • Image

不可用#

  • AssetBundle
  • Icon
  • RawImage

Async#

可用#

  • FutureBuilder
  • StreamBuilder

Basic#

可用#

  • Column
  • Container
  • Row
  • Text

不可用#

  • Appbar
  • ElevatedButton
  • FlutterLogo
  • Placeholder
  • Scaffold

Input#

全部 Input 组件均不可用，请使用 EditableText 组件创建输入框。

Interaction model#

可用#

  • AbsorbPointer
  • GestureDetector 仅 onTap 可用
  • IgnorePointer
  • Navigator

不可用#

  • Dismissible
  • DragTarget
  • Draggable
  • DraggableScrollableSheet
  • LongPressDraggable
  • Scrollable

Layout#

可用#

  • Align
  • AspectRatio
  • Center
  • ConstrainedBox
  • Container
  • Expanded
  • Offstage
  • Padding
  • SizedBox
  • Transform
  • Column
  • Flow
  • GridView
  • ListView
  • Row
  • Stack
  • Table
  • Wrap
  • CustomScrollView
  • SliverGrid
  • SliverList
  • SliverPadding
  • SliverPersistentHeader
  • SliverToBoxAdapter

不可用#

  • Baseline
  • CustomSingleChildLayout
  • FittedBox
  • FractionallySizedBox
  • IntrinsicHeight
  • IntrinsicWidth
  • LimitedBox
  • OverflowBox
  • SizedOverflowBox
  • CustomMultiChildLayout
  • IndexedStack
  • LayoutBuilder
  • ListBody

Painting and effect#

可用#

  • ClipOval
  • ClipRect
  • CustomPaint
  • DecoratedBox
  • Opacity
  • Transform

不可用#

  • BackdropFilter
  • ClipPath
  • FractionalTranslation
  • RotatedBox

Scrolling#

可用#

  • CustomScrollView
  • GridView
  • ListView

不可用#

  • DraggableScrollableSheet
  • NestedScrollView
  • NotificationListener
  • PageView
  • RefreshIndicator
  • ReorderableListView
  • ScrollConfiguration
  • Scrollable
  • Scrollbar
  • SingleChildScrollView

Styling#

可用#

  • MediaQuery
  • Padding

不可用#

  • Theme

Text#

在微信小程序中，RichText 中的 GestureRecongizer 不可用。

可用#

  • DefaultTextStyle
  • RichText
  • Text
